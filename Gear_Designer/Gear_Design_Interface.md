# Python 齒輪設計與視覺化工具
* [檔案位置](Gear_Design_Interface.py)
本程式是一個使用 **Python** 開發的 **漸開線齒輪設計與視覺化工具**，整合了：

* 齒輪幾何計算
* 極座標齒形繪製
* Tkinter 圖形介面
* Matplotlib 動畫顯示

使用者可以透過 GUI 輸入齒輪參數，即時生成 **漸開線齒輪草圖與旋轉動畫**，方便進行齒輪設計與幾何驗證。

## 系統架構

程式主要由三個模組構成：

```
Gear_Designer/
│
├─ gear_parameters.py     # 齒輪幾何公式
├─ data.py                # 參數儲存與表格
├─ graphics_generator.py  # 齒輪繪圖與動畫
```

主程式負責：

* GUI 操作
* 參數輸入
* 呼叫繪圖模組

整體流程如下：

```
使用者輸入參數
      ↓
資料表更新
      ↓
齒輪幾何計算
      ↓
Matplotlib 繪製齒輪
      ↓
GUI 顯示結果
```

## 齒輪幾何模型

程式使用 **漸開線齒輪模型 (Involute Gear)**。齒輪輪廓由四個部分組成：

| 函式           | 功能       |
| ------------ | -------- |
| line()       | 齒根到基圓的直線 |
| involute()   | 漸開線齒形    |
| topland()    | 齒頂圓弧     |
| bottomland() | 齒根圓弧     |
| hole()       | 中心孔      |
顏色區分：

* 藍色：漸開線齒形
* 綠色：齒側直線
* 紅色：齒頂與齒根
齒形曲線使用 **漸開線 (Involute curve)** 計算：

$$\phi = \sqrt{\left(\frac{R}{R_b}\right)^2 - 1}$$

## 圖形使用者介面 (GUI)
### 1️⃣ 主要參數設定

使用者可以輸入：

* 齒數
* 模數

程式會自動計算齒輪幾何。

---

### 2️⃣ 進階齒輪設定

可以調整：

* 齒頂倍率
* 齒根倍率
* 齒間倍率
* 壓力角
* 齒根圓角

這些參數會即時更新齒輪幾何。

---

### 3️⃣ 即時齒輪草圖

按下

```
齒輪草圖
```

即可生成齒輪輪廓。

圖形使用 **Matplotlib Canvas 嵌入 Tkinter**。

---

### 4️⃣ 齒輪動畫

按下

```
動畫
```

即可觀看齒輪旋轉動畫。動畫在 GUI 中顯示約 **5 秒後自動停止**。
